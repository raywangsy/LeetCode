### 解题思路：
- 双指针

```java
class Solution {
    public void sortColors(int[] nums) {
        int l = 0;
        int r = nums.length - 1;
        int p = 0;
        while (p <= r) {
            if (nums[p] == 0) {
                swap(nums, l++, p++);
            } else if (nums[p] == 1) {
                p++;
            } else {
                swap(nums, r--, p);
            }
        }
    }
    private void swap(int[] nums, int m, int n) {
        int temp = nums[m];
        nums[m] = nums[n];
        nums[n] = temp;
    }
}
```
